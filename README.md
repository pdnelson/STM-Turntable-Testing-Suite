# STM-Turntable-Testing-Suite
This is a testing suite intended to troubleshoot, test, and plan features for Statimatic turntables. Once connected, you will be able to control your turntable from your computer! Whether that be sending it commands, reading data from it, logging data from it over time, or any number of things. The possibilities are endless!

All commands sent through this software are logged, making it easy to trace back events if something goes awry.

## Connection Instructions
- Connect your Statimatic turntable to your computer via USB
- Launch this software
- Verify your turntable shows up on the list of COM ports
- Click "Connect"
- If the log says "Connection successful," then you've successfully connected to your turntable, and are ready to start executing commands!

## Simple View
Simple View is the simplest test tab: You can execute commands, one at a time, manually. This is primarily useful for testing a brand new command, before advancing functionality to the other tabs.
